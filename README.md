# Ripple CLI Test App

This repository is a small Ripple CLI test app built around Vite. It includes a `ripplex` command for running the development server, building the app, and serving production output.

## Install dependencies

```sh
pnpm install
```

## Run the dev server

Start the Vite-powered development server with:

```sh
vite
```

You can also run the included CLI directly for development:

```sh
pnpm exec ripplex dev
```
